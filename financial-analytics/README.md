These dashboards cover a wide array of financially related data. All client information has been ommited.

- The first dashboard is a snippet from an overall Quarterly report. It lays out each company's adjustment rates while also displaying industry averages at the time so that companies can see how they fair to the average.
- The second dashboard depicts a high-level overview of several metrics. These metrics include overhead costs, COGS, new vs returning customer metrics, etc.
- The third dashboard depicts a simplified Profit/Loss statement for a specified date range. In this case, we were looking at data from 2-weeks ago, 1-week ago, and last month. The Top 5 expenses recorded for that time period are also clearly displayed for the client.
- The fourth dashboard breaks out daily revenue/spend, and compared the actual amount earned/spent to what was projected in the client's budget. Also depicted is how current trends compare to last month's.
- The final dashboard depicted was an excercise on reimagining a Profit/Loss statment using visualizations. You will find a wide array of visualization types, including tree maps, donut charts, and bar charts.
